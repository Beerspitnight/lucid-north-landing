How It Works

A transparent look at our processes, from daily workflows to the core technology.

Onboarding Process

Getting started is simple. We securely upload your master schedule and calendar to populate the database, making the daily process seamless.

Daily Coverage Workflow

Each day, the administrator uploads the absence report. Our AI engine then runs a 4-tier scheduling process to ensure optimal coverage, highlighting any unfilled periods on the dashboard.

Core Architecture

Built on a reliable and scalable stack using Flask, SQLAlchemy, and OpenAI's GPT-4o API for powerful AI integration.

Data Workflow

Data flows from PDF absence reports through our OpenAI integration, which returns structured JSON for populating the database and generating coverage plans.

Security Posture

Security is foundational. We employ measures like CSRF protection and SQL injection prevention, with planned enhancements for SSO, data encryption, and more.
